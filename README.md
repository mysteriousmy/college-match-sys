# 高校运动会比赛小程序

微信小程序+云开发

功能：

- 普通用户查看新闻动态、排行榜、评论、关注比赛
- 运动会比赛用户查看管理自身在系统的信息，报名参赛，查询成绩。 \* 管理员，管理所有东西。
